# Zukko bilan Informatika — 1-sinf

Boshlang'ich sinf (1-sinf) o'quvchilari uchun **"Informatika va axborot texnologiyalari"** darsligi asosidagi o'yinli ta'lim platformasi.

## Xususiyatlari

- **33 ta interaktiv dars** — 4 bob (Sezgi a'zolari, Ketma-ketlik, Tomonlar va yo'nalishlar, Samaradorlik)
- **6 xil o'yin turi** — o'rganamiz, moslashtir, viktorina, ortiqchasini top, ketma-ketlik, rost/yolg'on
- **Uch tilli** — o'zbek, rus, ingliz (ovoz bilan)
- **Ovozli o'qish** — har bir topshiriq ovoz chiqarib o'qiladi
- **Jonli (animatsion) ikonlar** — Lottie / Noto Animated Emoji
- **Yulduz va medallar** — bola natijasi qurilmada saqlanadi
- **Ortga qaytish, ovozni o'chirish** va boshqa qulayliklar

## Texnologiya

Sof HTML + CSS + JavaScript (framework yo'q, build kerak emas, offlayn ishlaydi).
Ovoz: Aisha AI (o'zbek TTS) + brauzer Web Speech (rus/ingliz).

## Ishga tushirish

Oddiy statik sayt — istalgan veb-server bilan ochiladi:

```bash
python -m http.server 8000
# keyin brauzerda: http://localhost:8000
```

Yoki `index.html` ni to'g'ridan-to'g'ri oching.
